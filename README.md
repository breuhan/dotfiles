## Installation

From your homedirectory (on Linux/Mac OSX):

* `cd ~`
* `git clone https://github.com/cylonbrain/dotfiles.git ~/.dotfiles`
* `cd .dotfiles`
* `chmod +x makesymlinks.sh`
* `./makesymlinks.sh`
* `git submodule init`
* `git submodule update`

## Update
* `git pull`
* `git submodule foreach git pull origin master`
